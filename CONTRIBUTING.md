# Contributing to Meaning of Life — Open Project

Thank you for taking the time to contribute.

This project is built on simple, honest input from people around the world. Your perspective matters.

---

## How to Contribute

1. Fork this repository  
2. Open the `responses.md` file  
3. Add your entry at the end of the file  
4. Commit your changes  
5. Submit a Pull Request
