## 🧠 About the Project

Across cultures, generations, and experiences, people have always asked:

**Why do we exist?**

This repository is a growing archive of personal answers — contributed by individuals from around the world.

No single definition.  
No fixed truth.  
Only perspectives.

---

## 🎯 Objective

To build a public, structured collection of how people understand:
- Meaning of life  
- Purpose of existence  
- Personal philosophy  

Each entry adds a new lens to a shared human question.

---

## ✍️ How to Contribute

Contributing takes less than 2 minutes:

1. Fork this repository  
2. Open `responses.md`  
3. Add your perspective  
4. Submit a Pull Request
